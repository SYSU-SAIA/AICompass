# 复变函数
## 如何认识这门课

复变函数本质上讲是高等数学从实数域到复数域的扩展，实际生活中经常用于信号处理方面的学科。学习过程中注意求同存异，利用高数知识理解复变知识，注重理解复变比高数复杂的那一部分。

复变的体系与高数基本相似，无非是复数和复变函数的概念，微分、积分、级数的证明与计算，除此之外还有没接触过的留数和共形映射，而这两个概念分别建立在级数和映射之上，相信你很快就能理解并掌握。不要因为复数就感到害怕，你完全可以搬用高数的体系或者对其进行扩充，而不必从零建立框架。

复变函数对高数基础有一定要求，有很多证明或应用会省略高数中学过的定理，如果你对某个过程不理解，建议你翻阅高数的笔记寻找答案。在这个过程中，你既可以巩固高数，又能加深对复变函数的理解。一旦你理解而不是记住了某个结论，即使你在学期中忘记，在学期末也很容易拾起。
## 复习资料及应试策略

1. 复习资料：谷德峰老师的ppt。老师的ppt很全面，不过要注意细微的错误，注重对知识的理解。
2. 建议平时就对知识有较好的掌握，复变的内容比较细碎，理解不深刻或运用次数过少都会导致期末复习非常痛苦。
3. 学期末可以过一下ppt上所有的例题和习题，如果你能坚持自己做完ppt的习题，相信你一定可以拿到满分！
